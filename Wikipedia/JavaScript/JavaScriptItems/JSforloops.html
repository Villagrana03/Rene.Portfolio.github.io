For Loop

A for loop is a programming construct used for repetitive execution of a block of code.

for( let i = 0; i < array.length; i++)

for(Initialization; Condition; Iteration)

For loop Parameters

the for loop takes 3 parameters, which are for(initialization; condition; iteration)

  1. Initialization: The initialization parameter is executed before the loop begins. It is used to initialize the loop variable or any other necessary variables. This parameter typically assigns an initial value to the loop variable.
  2. Condition: The condition parameter is evaluated before each iteration of the loop. If the condition is true, the loop continues executing. If the condition is false, the loop terminates
  3. Iteration: The iteration parameter is executed at the end of each iteration, just before evaluating the condition again. It typically modifies the loop variable

array.forEach()

const numbers = [1,2,3,4,5]

numbers.forEach((n, index)=>{
console.log(`Element at index ${index} is: ${n}`)
})
